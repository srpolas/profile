# MD. Shahinur Rahman Polas — Portfolio

A modern, single-file portfolio website built with HTML, CSS, and JavaScript.

## Live Site

- **Domain:** https://srpolas.bd
- **Hosted on:** Cloudflare Pages

## Features

- Dark/Light theme toggle
- Animated particle network background
- Downloadable resume button
- GitHub profile integration
- Responsive design

## Project Files

| File | Description |
|------|-------------|
| `index.html` | Main portfolio page (HTML, CSS, JS in one file) |
| `profile.jpg` | GitHub profile picture |
| `CV of POLAS RampsBD+Silkcity.doc` | Downloadable resume/CV |

## Local Preview

Open `index.html` directly in any web browser.

## Deployment

This site is deployed via **Cloudflare Pages**, connected to the `main` branch of this GitHub repository.
